# QuizNest – Система интерактивного обучения

Это полноценное веб-приложение для создания, прохождения и соревнования в интеллектуальных квизах, построенное с использованием современных веб-технологий 

## Используемые технологии:
- Фронтенд. React, Vite, PWA (Service Workers)
- Бэкенд. Node.js, Express.js
- База данных. MongoDB
- Аутентификация. JWT
